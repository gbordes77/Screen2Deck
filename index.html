🔄 OCR Processing Pipeline

1. Image Upload → Validation 2. Preprocessing → 4 variants (Original, Denoised, Binarized, Sharpened) 3. EasyOCR → Multi-pass processing with 85% confidence threshold 4. Confidence Check → Optional Vision API fallback if <62% 5. Scryfall Validation → Card name verification via API 6. Export → Multiple formats (MTGA, Moxfield, Archidekt, TappedOut)
